快捷键(command+a后)

我的控制键为：Ctrl+a
  1. s：查看当前会话
  2. w：显示当前会话的所有窗口
  3. t：查看时间
  4. ,：窗口重命名
  5. o：选择pane，轮训选择，逆时针，也可方向键。
  6. ;：last pane
  7. q：输入序号选择
  8. !：使单独一个pane作为新的window
  9. 空格：更换pane排版
  10. ctrl+o：按顺序移动pane位置
  11. { ：往左、往上移动pane
  12. } ：往右、往下移动pane
  13. ：setw synchronize-panes(off)：同步所有窗口执行命令
  14. 复制模式：（需要添加配置）
    • ctrl+a [
    • h、j、k、l 上下左右
    • alt + 上下箭头：一屏跳
    • 空格：选择，回车完成复制到剪贴板
    • ctrl+a ] ：粘贴
