发表于 2017-08-24

基础整理

操作

  • mac安装
    1. brew update
    2. brew install mongodb
  • 启动
    1. mongod
    2. mongo
  • 关闭
    1. ctrl+c
    2. mongo
      > use admin db
      > db.shutdownServer({force: true})
  • 修复：
    1. 删除mongo/mongod.lock
    2. mongod -f /etc/mongod.conf –repair
    3. 另一教程里的：mongod –repair –dbpath /mongodb/ –repairpath /mongodb/repair/

语法

  • update:
    1. false 如果不存在，不插入
    2. true 查出来的记录都更新
  • $gt 大于
  • $lt 小于
  • $gte 大于等于
  • $lte 小于等于

样例:

大于0219小于0320
