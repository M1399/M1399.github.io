基础语法

  • 注释 行注释// 段注释/ /
  • == 会自动转换类型，一致后再比较
  • === 不会自动转换类型，类型不一致返回false
  • NaN 与所有值都不相等，包括自己

    NaN === NaN; //false
    用isNaN()函数判断
    isNaN（NaN）; //true

  • null：表示空的值（与python的None类似 ）
  • undefined: 表示未定义，在判断函数参数是否传递的情况下有用
  • 变量名：大小写英文、数字、$和_的组合
