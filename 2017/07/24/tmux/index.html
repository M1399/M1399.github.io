快捷键(command+a后)

我的控制键为：Ctrl+a
  1. s：查看当前会话
  2. w：显示当前会话的所有窗口
  3. t：查看时间
  4. ,：窗口重命名
  1. o：选择pane，轮训选择，逆时针，也可方向键。
  2. ;：last pane
  3. q：输入序号选择
  4. !：使单独一个pane作为新的window
  5. 空格：更换pane排版
  6. ctrl+o：按顺序移动pane位置
  7. { ：往左、往上移动pane
  8. } ：往右、往下移动pane
  9. ：setw synchronize-panes(off)：同步所有窗口执行命令
  10. 复制模式：（需要添加配置）
    • ctrl+a [
    • h、j、k、l 上下左右
    • alt + 上下箭头：一屏跳
    • 空格：选择，回车完成复制到剪贴板
    • ctrl+a ] ：粘贴
