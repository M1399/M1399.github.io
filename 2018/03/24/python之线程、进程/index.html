线程

使用

  • thread是低级模块，对它进行封装后是threading
  • 全局锁（GIL）：在任意一个指定的时间，有且只有一个线程在运行
  • 多线程复杂度高，不建议使用

  • 互斥锁（加锁一定要释放，不然会造成死锁）

    mlock = threading.Lock()
    mlock.acquire() # 加锁
    mlock.require() # 释放锁

  • Rlock 可重入锁

    mlock = threadling.Rlock()
